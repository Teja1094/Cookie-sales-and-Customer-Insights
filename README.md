# Cookie-sales-and-Customer-Insights
Project Objective

To develop an interactive dashboard that analyzes sales, revenue, and customer behavior. The goal is to provide clear insights into business performance and support data-driven decision-making.

Project Overview

This project transforms raw sales data into meaningful insights using data modeling, DAX measures, and visualizations. It consists of two dashboards focusing on sales performance and customer analytics.

Tools & Technologies Used

Power BI was used for dashboard development, along with DAX for calculations and Power Query for data transformation. Python was integrated to create advanced custom visualizations.

Data Model

A star schema data model was implemented with a central sales fact table connected to dimension tables like date, product, and customer. This structure ensures efficient filtering and accurate analysis.

Conclusion

The dashboard provides a comprehensive view of sales and customer insights, enabling better business decisions. It can be further enhanced with advanced analytics and real-time data integration for deeper insights.
